# Agentic Pipeline Docs

This folder contains the human-facing documentation for the supervised agentic delivery system being designed in this repository.

## Purpose

The project aims to create a reliable end-to-end workflow that combines:

- `Linear` for product context, planning, approvals, and documentation
- `Claude Code` and/or `Cursor` for agent execution
- `GitHub` for code review, CI, merge, and deployment controls

The goal is not full autonomy. The goal is a supervised operating system where agents can move quickly, while humans remain accountable for approval, coordination, and review.
